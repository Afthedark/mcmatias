# Frontend Assets

Esta carpeta contiene recursos estáticos como logos, imágenes e iconos.

## Estructura sugerida:
```
assets/
├── logo.png          # Logo principal de MCMatias
├── favicon.ico       # Icono para el navegador
└── images/           # Otras imágenes del proyecto
```

## Uso en HTML:
```html
<img src="assets/logo.png" alt="MCMatias Logo">
```
